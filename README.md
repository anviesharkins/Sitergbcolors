# RGB Colors

Site simples e original para visualizar, pesquisar e copiar valores de cores em RGB e HEX.

## Funções

- Pesquisa por nome da cor, RGB ou HEX.
- Lista de cores populares separadas por categoria.
- Gerador de cor RGB com sliders.
- Botão para copiar RGB e HEX.
- Navegador de todas as cores RGB usando páginas leves de 256 cores.

## Como publicar no GitHub Pages

1. Abra o repositório no GitHub.
2. Vá em **Settings**.
3. Entre em **Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Em **Branch**, escolha `main` e a pasta `/root`.
6. Salve.

Depois de alguns segundos ou minutos, o GitHub vai gerar o link do site.
